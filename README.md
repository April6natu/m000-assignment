# m000-assignment

This is my M000 assignment project for the Aiken/Cardano workshop.

Run tests:

```sh
cd m000-assignment
# run all tests
aiken check
# run a single test (no angle brackets):
aiken check -m m000_trace
```

## What to include when submitting
- Personal details (name, email)
- GitHub repository URL
- Short description of the work and what you learned
- Notes about challenges and how you solved them
- Explanation of UTxO vs eUTxO

Fill these out in `SUBMISSION.md` before submitting.
